[KONTEKS FORM CHECKOUT & KERANJANG]
Halaman UI asal: checkout.html & keranjang.html
Pemetaan Request Form ke Backend Laravel Controller:
- Input Teks Area Catatan : name="order_notes"   (diambil dari id="order-notes" di keranjang)
- Input Nama Depan        : name="first_name"    (diambil dari id="first-name" di checkout)
- Input Nama Belakang     : name="last_name"     (diambil dari id="last-name" di checkout)
- Input Alamat Email      : name="email"         (diambil dari id="email" di checkout)
- Input Alamat Pengiriman : name="address"       (diambil dari id="address" di checkout)
- Input Kota              : name="city"          (diambil dari id="city" di checkout)
- Input Kode Pos          : name="postal_code"   (diambil dari id="postal-code" di checkout)
- Pilihan Metode Bayar    : name="payment"       (Value radio: 'Bank Transfer' atau 'QRIS')
